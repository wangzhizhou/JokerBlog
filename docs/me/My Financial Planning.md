# 个人基本情况

应税工资 = 税前收入 - 社保 - 公积金(5%-12% * 基数(年薪均分12个月)

个人所得税 = (应税工资 - 起征点）* 税率  - 速算扣

薪资计算方法: 税后收入 = 应税工资 - 个人所得税
